# CalculoNotaFacens
Fluxograma, Algoritmo em linguagem comum e Algoritmo em Java comentado.

### Inicio	
		Receber notas
	 	  Nota ac1
		  Nota ac2
		  Nota ag
		  Nota af
		Calcular o valor final das notas usando o peso
		     ac1 * 1.5
 		     ac2 * 3.0 
 		     ag * 1.0
 		     af * 4.5
		calcular a média final 
		  se a média for >5 
			o aluno esta aprovado
		  se for <5 
			o aluno esta reprovado
